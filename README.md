# Dynamic Login Page with OTP Verification 🔐

A frontend practice project built in my 2nd year, right after learning the fundamentals of HTML, CSS, and JavaScript. This repo brings together everything I picked up into hands-on, functional UI builds.

> 📌 Note: This repo actually contains **two projects** — a Login/Signup UI with OTP verification, and separate layout practice using Flexbox and CSS Grid.

## 🚀 Projects Included

### 1. Dynamic Login / Sign Up Page
A toggle-based authentication UI with:
- Sign In and Create Account panels with a sliding toggle animation
- Client-side form validation (name, email, password)
- Responsive, card-based layout

### 2. OTP Verification
- 4-digit OTP input with auto-focus to the next field on entry
- Backspace navigation back to the previous field
- Numeric-only input handling

### 3. Layout Practice — Flexbox & CSS Grid
- Standalone demos exploring Flexbox alignment/justification
- Standalone demos exploring CSS Grid structuring

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎯 What I Learned
This project was my first real step from "learning syntax" to actually building functional UI — handling form validation, OTP input logic, and responsive layouts using Flexbox and Grid.
